- Метрическое пространство

- Примеры

Чтобы в полной мере понять специфику ультраметрических пространств, логично начать изложение с такого базового понятия современной математики, как линейное (векторное) пространство. И, не сильно углубляясь в этом направлении, в дальнейшем перейти непосредственно к теме данной работы.

Пусть L - некоторое множество объектов произвольной природы, а R - поле действительных (или комплексных) чисел. Множество L называют линейным пространством, если на нем определены две операции: 1) операция сложения любых двух элементов этого множества и 2) операция умножения элементов этого множества на число, причем эти операции удовлетворяют некоторым естественным аксиомам.

Пусть теперь x1,x2,...,xn - некоторые элементы линейного пространства L (векторы), а c1,c2,...,cn - произвольные действительные (или комплексные) числа. Элемент пространства L, равный c1 * x1 + c2 * x2 + ... + cn * xn, называется линейной комбинацией элементов .

Система (набор) элементов пространства L называется линейно независимой, если линейная комбинация равна нулевому элементу пространства только в случае c1 = c2 = ... = cn = 0 .

Определение. Система элементов пространства L называется линейно зависимой, если равенство c1 * x1 + c2 * x2 + ... + cn * xn = 0 выполнено при некотором наборе констант с1, c2, ..., cn, хотя бы одна из которых отлична от нуля.

Таким образом, система называется линейно зависимой, если она не является линейно независимой.

 Линейное пространство имеет размерность n (или, коротко, n-мерно), если в нем найдется n линейно независимых элементов, но любые (n+1) элемент линейно зависимы. Линейное пространство называется бесконечномерным, если в нем можно указать любое наперед заданное число линейно независимых элементов.

Легко доказывается, что любой элемент n-мерного линейного пространства можно представить (причем единственным способом) в виде линейной комбинации произвольных n элементов, образующих линейно независимую систему. Этот факт весьма важен: в самом деле, n элементов линейно независимой системы в n-мерном пространстве оказываются теми "кирпичиками", из которых можно сложить абсолютно любой элемент пространства. В силу важности этого факта введем специальное понятие - базис линейного пространства.

Система элементов линейного пространства называется базисом этого пространства, если любой элемент этого пространства можно единственным образом представить в виде линейной комбинации элементов данной системы.

В конечномерном (n-мерном) пространстве любая линейно независимая система из n элементов образует базис. Естественно поставить вопрос: справедливо ли аналогичное утверждение для бесконечномерных пространств, и что вообще представляет собой базис бесконечномерного пространства? 
Изучение бесконечномерных линейных пространств (точнее, рассмотрение базисов в этих пространствах) невозможно лишь в терминах самого линейного пространства, но требует введения некой метрики. Таким образом, рассматриваемое пространство должно быть наделено не только линейной, но и метрической структурой. В этом заключается коренное отличие бесконечномерных линейных пространств от конечномерных.

Множество M называется метрическим пространством, если каждым двум элементам x, y этого множества поставлено в соответствие действительное число р(x,y), называемое расстоянием между элементами x и y, причем выполнены следующие аксиомы:
1. p(x,y)>=0 , причем  p(x,y)=0 <=> x=y
2. p(x,y)=p(y,x) , для любых x,y из М
3. p(x,y)<=p(x,z)+p(z,y) для любых x,y,z из М

Функция p : МхМ -> R называется метрикой пространства М.
Третья аксиома, носящая название "неравенство треугольника", имеет наглядную интерпретацию: длина любой из сторон треугольника не превосходит суммы двух других сторон (равенство достигается, если треугольник "вырожден").

В линейных пространствах наряду с метрикой используют понятие нормы элемента.
Линейное пространство L называется нормированным, если каждому элементу x этого пространства поставлено в соответствие действительное число ||x|| (норма x ), причем выполнены следующие аксиомы:
1. ||x||>=0 для любого x, причем ||x||=0 <=> x - нулевой вектор
2. ||c*x||=|c|*||x||  для любого x из L и c из R
3. ||x + y|| <= ||x|| + ||y||  для любых x, y из данного пространства.

Простейшими примерами нормированных пространств могут служить множества действительных чисел R и комплексных чисел C, где в качестве нормы числа рассматривается его модуль, а также пространство векторов на плоскости (или в пространстве) с нормой, равной длине вектора.
Любое линейное нормированное пространство можно сделать метрическим пространством естественным способом - ввести индуцированную метрику p(x,y) = ||x - y||.
Не составляет труда убедиться в выполнимости аксиом метрического пространства для данной метрики.
Обратное утверждение, вообще говоря, неверно: не в любом метрическом пространстве можно ввести норму, поскольку понятие нормы вводится лишь в линейном пространстве, а метрическое пространство может не быть наделено линейной структурой. Однако, если метрическое пространство наделено линейной структурой (является линейным пространством), то его всегда можно сделать нормированным, введя норму ||x|| = p(x,0)

Пусть теперь {xn} - некоторая последовательность элементов линейного нормированного пространства L, а x0 - некоторый фиксированный элемент L. Для каждого номера n найдем ||xn - x0||. Тем самым получим числовую последовательность {||xn - x0||}
Элемент х0 линейного нормированного пространства L называется пределом последовательности {xn}, если lim||xn - x0|| = 0 (или lim p(xn,x0) = 0)
Если необходимо, то указывают, по какой норме рассматривается предел, поскольку одна и таже последовательность может иметь предел по одной норме и не иметь - по другой.
Рассмотрим еще понятие фундаментальной последовательности, тесно связанное с понятием предела.
Последовательность {xn} элементов линейного нормированного пространства называется фундаментальной, если:
для любого E>0 существует такой номер N, что для любых n,m > N => ||xn - xm|| < E
Очевидно, что любая сходящаяся последовательность фундаментальна.
Из курса анализа известен критерий Коши: числовая последовательность сходится тогда и только тогда, когда она фундаментальна. Иными словами, пространство R устроено так, что в нем не только из сходимости следует фундаментальность, но и наоборот. Однако не любое линейное нормированное пространство устроено таким образом: например, в пространстве рациональных чисел Q (с обычными линейными операциями и нормой ||x|| = |x|) фундаментальная последовательность может расходиться (такая ситуация имеет место, если пределом последовательности рациональных чисел является число иррациональное).

Линейное нормированное пространство называется полным, если в нем любая фундаментальная последовательность сходится.
Полное линейное нормированное пространство называют также банаховым пространством (по имени выдающегося польско-украинского математика Стефана Банаха (1892-1945)).
Пространства R и C - банаховы, а пространство Q - нет. 

Пусть дано множество X. Система T его подмножеств называется топологией на X, если выполнены следующие условия:
1. Пустое множество и все множество Х принадлежит Т
2. Объединение произвольного семейства множеств, принадлежащих T, принадлежит T
3. Пересечение конечного семейства множеств, принадлежащих T, принадлежит T
Пара (X, T) называется топологическим пространством. Множества, принадлежащие T, называются открытыми множествами.
Открытое множество — это множество, каждый элемент которого входит в него вместе с некоторой окрестностью. Открытое множество является фундаментальным понятием общей топологии.

    Любое метрическое пространство обладает естественной топологией, базой для которой служит множество открытых шаров, то есть множеств следующего типа:
B(x; r) = {y ∈ M ∣ d(x, y) < r} ,
где x есть точка в M  и r — положительное вещественное число, называемое радиусом шара. Иначе говоря, множество является открытым, если вместе с любой своей точкой оно содержит открытый шар с центром в этой точке. 


Иерархии:
Для произвольного множества Х, будем обозначать Exp(X) - множество всех подмножеств Х.
Пусть Х - множество и Н - подмножество Exp(X) такое, что выполняются следующие условия:
1. Объединение множеств В = Х, где В - элементы множества Н
2. Произвольные В1, В2 из Н либо не пересекаются, либо одно из них включено в другое
3. Разность любых А, В из Н представляется в виде объединения некоторого (счетного?) семейства множеств из Н 
Тогда Н - называется иерархией, пара (Х, Н) - иерархическим пространством.
Если само Х также принадлежит Н, иерархия и иерархическое пространство называются ограниченными.
Нетрудно видеть, что Н - является базой топологии, причем каждое множество из иерархии является как открытым, так и замкнутым (?).
Иерархическое пространство (Х,Н) и иерархия Н называются хаусдорфовыми, если для любых различных элементов х, у из Х существуют А, В из Н такие, что х принадлежит А, у принадлежит В, а пересечение А и В - пустое множество.
Пусть Л - частично упорядоченное множество, (Х, Н) - ограниченное иерархическое пространство и f:H->Л строго убывающее сюръективное отображение. Назовем f функцией уровня, а значение f(В) для произвольного В из Н - уровнем множества В. Набор (Х, Н, Л, f) называют шкалированным иерархическим пространством, если для любого l из Л семейство прообразов Нl = f^(-1)(l) образует разбиение Х.

На практике существует большое количество споосбов построения иерархических классификаций. Принципиально различаются два метода: один основан на последовательных дихотомиях, начиная с полного множества Х (иерархическая нисходящая классификация), другой - на последовательных объединениях, начиная с отдельных элементов множества Х (иерархическая восходящая классификация).
Методы прямого построения иерархических классификаций могут эффективно применяться лишь к относительно небольшим объемам данных. 

Множество M называется метрическим пространством, если каждым двум элементам x, y этого множества поставлено в соответствие действительное число р(x,y), называемое расстоянием между элементами x и y, причем выполнены следующие аксиомы:
1. p(x,y)>=0 , причем  p(x,y)=0 <=> x=y
2. p(x,y)=p(y,x) , для любых x,y из М
3. p(x,y)<=p(x,z)+p(z,y) для любых x,y,z из М

Функция p : МхМ -> R называется метрикой пространства М.
Третья аксиома, носящая название "неравенство треугольника", имеет наглядную интерпретацию: длина любой из сторон треугольника не превосходит суммы двух других сторон (равенство достигается, если треугольник "вырожден").

В линейных пространствах наряду с метрикой используют понятие нормы элемента.
Линейное пространство L называется нормированным, если каждому элементу x этого пространства поставлено в соответствие действительное число ||x|| (норма x ), причем выполнены следующие аксиомы:
1. ||x||>=0 для любого x, причем ||x||=0 <=> x - нулевой вектор
2. ||c*x||=|c|*||x||  для любого x из L и c из R
3. ||x + y|| <= ||x|| + ||y||  для любых x, y из данного пространства.

Простейшими примерами нормированных пространств могут служить множества действительных чисел R и комплексных чисел C, где в качестве нормы числа рассматривается его модуль, а также пространство векторов на плоскости (или в пространстве) с нормой, равной длине вектора.
Любое линейное нормированное пространство можно сделать метрическим пространством естественным способом - ввести индуцированную метрику p(x,y) = ||x - y||.
Не составляет труда убедиться в выполнимости аксиом метрического пространства для данной метрики.
Обратное утверждение, вообще говоря, неверно: не в любом метрическом пространстве можно ввести норму, поскольку понятие нормы вводится лишь в линейном пространстве, а метрическое пространство может не быть наделено линейной структурой. Однако, если метрическое пространство наделено линейной структурой (является линейным пространством), то его всегда можно сделать нормированным, введя норму ||x|| = p(x,0)

Пусть теперь {xn} - некоторая последовательность элементов линейного нормированного пространства L, а x0 - некоторый фиксированный элемент L. Для каждого номера n найдем ||xn - x0||. Тем самым получим числовую последовательность {||xn - x0||}
Элемент х0 линейного нормированного пространства L называется пределом последовательности {xn}, если lim||xn - x0|| = 0 (или lim p(xn,x0) = 0)
Если необходимо, то указывают, по какой норме рассматривается предел, поскольку одна и таже последовательность может иметь предел по одной норме и не иметь - по другой.
Рассмотрим еще понятие фундаментальной последовательности, тесно связанное с понятием предела.
Последовательность {xn} элементов линейного нормированного пространства называется фундаментальной, если:
для любого E>0 существует такой номер N, что для любых n,m > N => ||xn - xm|| < E
Очевидно, что любая сходящаяся последовательность фундаментальна.

Иными словами, пространство R устроено так, что в нем не только из сходимости следует фундаментальность, но и наоборот. Однако не любое линейное нормированное пространство устроено таким образом: например, в пространстве рациональных чисел Q (с обычными линейными операциями и нормой ||x|| = |x|) фундаментальная последовательность может расходиться (такая ситуация имеет место, если пределом последовательности рациональных чисел является число иррациональное).

Линейное нормированное пространство называется полным, если в нем любая фундаментальная последовательность сходится.
Полное линейное нормированное пространство называют также банаховым пространством (по имени выдающегося польско-украинского математика Стефана Банаха (1892-1945)).
Пространства R и C - банаховы, а пространство Q - нет.

Иерархии:
Для произвольного множества Х, будем обозначать Exp(X) - множество всех подмножеств Х.
Пусть Х - множество и Н - подмножество Exp(X) такое, что выполняются следующие условия:
1. Объединение множеств В = Х, где В - элементы множества Н
2. Произвольные В1, В2 из Н либо не пересекаются, либо одно из них включено в другое
3. Разность любых А, В из Н представляется в виде объединения некоторого (счетного?) семейства множеств из Н 
Тогда Н - называется иерархией, пара (Х, Н) - иерархическим пространством.
Если само Х также принадлежит Н, иерархия и иерархическое пространство называются ограниченными.
Нетрудно видеть, что Н - является базой топологии, причем каждое множество из иерархии является как открытым, так и замкнутым (?).
Иерархическое пространство (Х,Н) и иерархия Н называются хаусдорфовыми, если для любых различных элементов х, у из Х существуют А, В из Н такие, что х принадлежит А, у принадлежит В, а пересечение А и В - пустое множество.
Пусть Л - частично упорядоченное множество, (Х, Н) - ограниченное иерархическое пространство и f:H->Л строго убывающее сюръективное отображение. Назовем f функцией уровня, а значение f(В) для произвольного В из Н - уровнем множества В. Набор (Х, Н, Л, f) называют шкалированным иерархическим пространством, если для любого l из Л семейство прообразов Нl = f^(-1)(l) образует разбиение Х. В случае, если Л представляет из себя множество натуральных чисел с нулем и обычным отношением порядка, (Х, Н, f) - дискретно шкалированное иерархическое пространство.

На практике существует большое количество споосбов построения иерархических классификаций. Принципиально различаются два метода: один основан на последовательных дихотомиях, начиная с полного множества Х (иерархическая нисходящая классификация), другой - на последовательных объединениях, начиная с отдельных элементов множества Х (иерархическая восходящая классификация).
Методы прямого построения иерархических классификаций могут эффективно применяться лишь к относительно небольшим объемам данных. 

Пусть Х - некоторое множество. Пара (Х, d) называется ультраметрическим (или неархимедовым) пространством, если каждым двум элементам x, y этого множества поставлено в соответствие неотрицательное действительное число d(x,y), называемое расстоянием между элементами x и y, причем выполнены следующие аксиомы:
1. d(x,y)>=0 , причем  d(x,y)=0 <=> x=y
2. d(x,y)=d(y,x) , для любых x,y из X
3. d(x,y)<=max[d(x,z),d(z,y)] для любых x,y,z из X
d - называют ультраметрикой, неархимедовой метрикой или неархимедовым расстоянием. Третья аксиома называется усиленным неравенством треугольника или ультраметрическим неравенством. Очевидно, что согласно своему названию, ультраметрическое неравенство сильнее неравенства треугольника, фигурирующего в определении метрического пространства, так что всякое ультраметрическое пространство является метрическим.
Как и в теории метрических пространств, рассматривается ослабленный вариант условия 1, когда для различных элементов расстояние может быть нулевым. Такие пространства называют псевдоультраметрическими или неархимедовыми псевдометрическими. Они порождают нехаусдорфову топологию.

Свойства:
Свойство 1.
Все треугольники в ультраметрическом пространстве - равнобедненные, у которых основание не больше боковой стороны (в частности - равносторониие).
Действительно, если для некоторых элементов x,y,z из Х расстояния между ними попарно различны, это значит, что для большей из них усиленное неравенство треугольника несправедливо.

Свойство 2.
Два шара в ультраметрическом пространстве либо не пересекаются, либо один из них включен в другой. В частности, два замкнутых (либо два открытых) шара одинакового радиуса либо не пересекаются, либо совпадают.
Пусть В1 - шар радиуса r1 с центром в точке х, В2 - шар радиуса r2 с центром в точке y. И пусть r1<=r2.
Допустим, что В1 и В2 - пересекаются, т.е. существует такой k из Х, что d(x,k) <= r1 и d(y,k) <= r2.
Покажем, что В1 лежит целиком в В2. Возьмем произвольную точку m из В1 и рассмотрим точки x, k, m:
d(k,m) <= max[d(x,k), d(x,m)] по определению шара В1. Но тогда d(y,m) <= max[d(y,k), d(k,m)] <= r2, т.е. m лежит в В2. Что и требовалось доказать.

Свойство 3.
Любая точка шара в ультраметрическом пространстве является его центром.
Возьмем шар В1 радиуса r с центром в точке x и принадлежащей ему точкой у. Рассмотрим шар В2 радиуса r с центром в y. Если некоторая точка k принадлежит шару В1, то d(у,k) <= max[d(x,y),d(x,k)], т.е. k принадлежит шару В2. И наоборот, если некоторая точка k принадлежит шару В2, то d(x,k) <= max[d(x,y), d(y,k)] и, следовательно, k принадлежит шару В1. Что и требовалось доказать.

Свойство 4.
В ультраметрическом пространстве каждый шар (как открытый, так и замкнутый) является открытым и замкнутым множеством.

Свойство 5.
Для компактного ультраметрического пространства и для любого r > 0 существует и единственно конечное покрытие этого пространства непересекающимися открытыми (замкнутыми) шарами радиуса r.

Свойство 6.
Пусть (X,d) - неархимедово метрическое пространство и множество Y является подмножеством Х. Если Х конечно, то множество D(Y,d) = {d(x,y)|x, y - различные элементы из Y} также конечно. В противном случае D(X) = {r1, r2, ...}, где rk стремится к нулю. Другими словами, ноль является единственной предельной точкой D(X).

Замечание.
В этих обозначениях в бесконечном ультраметрическом пространсте всякий открытый шар радиуса r(k) - это замкнутый шар радиуса r(k+1), а замкнутый шар радиуса r0 - это все пространство Х

Свойство 7.
Пусть (X,d) - компактное неархимедово метрическое пространство, D(X,d) = {r0, r1, ...}, ri->0. Любой шар радиуса r в (X,d) совпадает с некоторым шаром радиуса ri из D(X,d), где i определяется из условия: ri = max[rj|rj <= r]

Свойство 8.
Пусть (X,d) - компактное неархимедово метрическое пространство, r >= s > 0. Любой шар радиуса r единственным образом разбивается на конечное число шаров радиуса s.

В ультраметрическом пространстве каждое из множеств: множество всех шаров, множество замкнутых шаров и множество открытых шаров, - образует иерархию. Каждый шар - как открытый, так и замкнутый - является как открытым, так и замкнутым множеством. В компактном ультраметрическом пространстве каждый открытый шар радиуса r(i) является замкнутым шаром радиуса r(i+1). Для любого r > 0 все замкнутые (а также все открытые) шары радиуса r образуют конечное разбиение Х. Компактное ультраметрическое пространство является дискретно шкалированным иерархическим пространством, где слой Н(k) образуется всеми замкнутыми (или всеми открытыми) шарами радиуса r(k).
